# Laravel Docs
--------------

### Generation
```
# navigate to the folder root
cd path/to/repo/

# install deps
npm install

# navigate to laravel docs root
cd laravel/

# main script
node main.js

# choose from `4.0.x`, `4.1.x`, etc
```


### Development

##### To generate docs, you need to fetch submodule into your repo
```
git submodule update --init
```
